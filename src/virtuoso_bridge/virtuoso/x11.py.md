# X11 对话框检测和关闭

## 功能概述

当模态对话框阻塞 Virtuoso CIW 事件循环时，通过 SSH 直接操作 X11 来检测和关闭对话框，绕过 SKILL 通道。

## 问题背景

当模态对话框阻塞 CIW 时，所有 `execute_skill()` 调用都会超时。本模块使用直接 SSH + 远程 Python2/Xlib 来查找和关闭对话框，不接触 SKILL 通道。

## 主要函数

### `find_dialogs(runner, user, display=None) -> list[dict[str, Any]]`

在远程 X11 显示器上查找阻塞对话框窗口。

**参数**:

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `runner` | `SSHRunner` | - | SSH 运行器 |
| `user` | `str` | - | 远程用户名 |
| `display` | `str \| None` | `None` | X11 显示（默认自动检测） |

**返回**: `list[dict]` - 对话框列表，包含 `window_id`, `title`, `x`, `y`, `w`, `h`

```python
from virtuoso_bridge.virtuoso.x11 import find_dialogs

dialogs = find_dialogs(ssh_runner, "zhangz")
for d in dialogs:
    print(f"Dialog: {d['title']} at ({d['x']}, {d['y']})")
```

### `dismiss_dialogs(runner, user, display=None) -> list[dict[str, Any]]`

查找并关闭所有阻塞对话框窗口。

**返回**: `list[dict]` - 结果列表（找到的对话框 + 关闭结果）

```python
from virtuoso_bridge.virtuoso.x11 import dismiss_dialogs

results = dismiss_dialogs(ssh_runner, "zhangz")
for r in results:
    print(f"Dismissed: {r.get('title', 'unknown')}")
```

## 显示解析

显示优先级：
1. 显式参数 `display`
2. 环境变量 `VB_DISPLAY`
3. 自动检测（返回 None）

## 辅助脚本

需要远程存在 `x11_dismiss_dialog.py` 脚本，位于 `virtuoso_bridge/resources/` 目录。

首次调用时自动上传到 `/tmp/virtuoso_bridge_<user>/x11_dismiss_dialog.py`。

## 依赖

- 远程主机需要 Python2 和 Xlib (`python2`, `from Xlib import X, display`)
- 需要有效的 X11 会话
